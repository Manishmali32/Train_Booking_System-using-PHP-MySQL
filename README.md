# Train_Booking_System-using-PHP-MySQL
A web-based Train Booking System developed using PHP and MySQL that allows users to search trains, check availability, book tickets, and manage bookings.




## Features
- User registration and login
- Search trains by source and destination
- View train schedules
- Check seat availability
- Book train tickets
- Cancel booked tickets
- View booking history
- Admin panel to manage trains and schedules

---

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Apache Server (XAMPP)

---

## Database Tables
- users
- trains
- bookings
- seats
- admin

---

## How to Run the Project
1. Install **XAMPP**
2. Start **Apache** and **MySQL**
3. Clone or download this repository
4. Import the database file into **phpMyAdmin**
5. Copy the project folder into the `htdocs` directory
6. Open your browser and go to:  
   `http://localhost/train-booking-system`
